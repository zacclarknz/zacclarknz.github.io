Zac Clark.

I'm an experienced data analyst who has worked in both the public and private sector. This is place to showcase some of publicly available recent work.

Projects

  • The urban developmnent capacity dashboard. This tool was built for councils to help monitor the housing market. It includes a number of simple indicators, includng a quality adjusted house price index, Sales Price Appraisal Ratio (SPAR). I also created one of the price efficiency indicators, a measure of competitiveness in the developable land market called the land concentration of control measures. This includes an innovative application of the Herfindahl-Hirschman index (HHI) to the urban land market. See the technical guide I wrote here.

  • Report into foreign buyers in New Zealand

    I looked into the recent data released by Stats NZ on the activity of foreign buyers in property transfers. Includes a R generated html report with interactive chloropeth. Code here.

  • Dashboard of Wellington Sculptures

    I used R to process geospatial data, then a combination of javascript libraries are used to create dynamic crossfiltered maps and charts. When you change one chart all the other charts dynamically change. Code here.
