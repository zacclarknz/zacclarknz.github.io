Zac Clark

I'm an data scientist with over four years experience in both the public and private sector. This website showcases some of my recent work.

Projects

  • The urban development capacity dashboard.

    This tool was built for local authorities to help them monitor the housing market. It includes a number of simple indicators I built, includng a quality adjusted house price index: the Sales Price Appraisal Ratio (SPAR). I also created one of the price efficiency indicators, a measure of competitiveness in the developable land market: the Land Concentration Control Measures. This includes an innovative application of the Herfindahl-Hirschman index (HHI) to the urban land market. In order to create the measure I had to combine data from across many different agencies. See the technical guide method section I wrote here.

  • Report into foreign buyers in New Zealand

    I used R to generate a short HTML report of the recent data released by Stats NZ on the activity of foreign buyers in property transfers. This includes an R generated HTML report with an interactive heat map (chloropeth). Code here.

  • Dashboard of Wellington sculptures (Chrome tested)

    I used R to process geospatial data, then a combination of javascript libraries were used to create dynamic crossfiltered maps and charts. When you change one chart all the other charts and data-table dynamically change. Code here.
